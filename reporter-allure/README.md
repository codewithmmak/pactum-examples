# Allure reporter

Example of integrating `mocha-allure-reporter` & `allure-commandline` with pactum.

## Get Started

Prerequisites
* JDK should be installed and JAVA_HOME path should be set

Installation

```sh
npm i
```

Running Tests

```sh
npm test
```

Generate Allure Report

```sh
npm run allure-report-gen
```
